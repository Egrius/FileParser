## 📁 FileParser

**Многопользовательское веб-приложение** для загрузки, хранения и анализа текстовых файлов. Каждый пользователь работает только со своими файлами, обеспечивая полную изоляцию данных.

> 🎯 Проект создан в учебных целях для отработки: многопользовательской архитектуры, изоляции данных, Stream API для анализа текста, регулярных выражений и событийной модели.

## ✨ Возможности
### 👥 Для пользователей

- Регистрация и Basic-аутентификация
- Загрузка текстовых файлов (до 10 МБ)
- Просмотр списка загруженных файлов с пагинацией
- Просмотр содержимого файла
- Удаление файлов (с подтверждением паролем)

### 📊 Анализ текста

- Топ слов — подсчёт частоты слов с исключением стоп-слов
- Статистика по первой букве — группировка слов по начальной букве
- Анализ пунктуации — подсчёт знаков препинания
- Длина слов — слово -> его длина

### 🔍 Regex-поиск

- Email'ы — поиск всех email-адресов
- Телефоны — поддержка форматов: +375 (29) 123-45-67, +375 33 9876543
- IP-адреса — IPv4 адреса
- Даты — в формате ДД.ММ.ГГГГ

### 🔐 Безопасность

- Basic-аутентификация через Spring Security
- Изоляция данных — пользователь видит только свои файлы
- Валидация входящих DTO
- Глобальный обработчик исключений с понятными сообщениями

### 📊 Работа с БД

- PostgreSQL — основная БД
- Spring Data JPA — репозитории
- UUID в качестве первичных ключей
- Кастомные запросы с @Query
- Пагинация для списков файлов

### 🧪 Тестирование

Проект покрыт тестами на всех уровнях:

- Unit-тесты — сервисы с Mockito
- Интеграционные тесты — с реальной БД (PostgreSQL)
- Тесты контроллеров — MockMvc
- Тесты репозиториев — @DataJpaTest

## 🚀 Быстрый старт
### 🔧 Требования

- Java 21+
- PostgreSQL
- Maven

###  Установка и запуск
``` bash
# 1. Клонировать репозиторий
git clone https://github.com/Egrius/FileParser.git
cd FileParser/app

# 2. Создать базу данных
createdb fileparser

# 3. Настроить подключение в application.properties
#    (можно скопировать application.properties из проекта)

# 4. Собрать проект
./gradlew clean build

# 5. Запустить (имя JAR может отличаться)
java -jar build/libs/fileparser.jar
```
